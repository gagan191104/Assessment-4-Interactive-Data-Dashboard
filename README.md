## Assessment 4
The Interactive Data Dashboard is developed as part of Assessment 4 to demonstrate the use of JavaScript, DOM manipulation, and API integration. The application allows users to interact with dynamic data such as generating random users, checking weather information, managing tasks, and filtering product lists.

# Features

1. Random User Generator – Fetches user data from an API and displays profile picture, name, email, and country.
2. Weather Search – Allows users to search weather details like temperature and wind speed.
3. Dynamic To-Do List – Users can add, delete, and mark tasks as completed.
4.Live Search Filter – Instantly filters a list of products based on user input.

# Technologies Used

HTML
CSS
JavaScript
DOM Manipulation
API Integration

# APIs Used

Random User API – https://randomuser.me/api/
Open Meteo Weather API – https://api.open-meteo.com
